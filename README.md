### Theme

### Useful settings
```
set relativenumber
set nohlsearch
set tabstop=4 softtabstop=4
set hidden
set incsearch
set scrolloff=8
set noswapfile
set nobackup
set undodir=~/.vim/undodir
set undofile
```
### For terminal color consistency
https://github.com/gruvbox-community/gruvbox
```
colorscheme gruvbox
set background=dark
highlight Normal ctermbg=none 
```
